# Kullanım Rehberi: Portal

Portal, Jun‑Oro uygulamalarına erişim için ana sayfadır. Kartlara tıklayarak ilgili subdomain’e gidersiniz.

Adımlar:
1) tarayıcıdan `https://jun-oro.com` adresine gidin.
2) İlgili uygulama kartını tıklayın.
3) Admin veya özel alanlar için erişim izni (Cloudflare Access) gerekebilir.

Not: Bazı uygulamalar “yakında” durumunda olabilir.
