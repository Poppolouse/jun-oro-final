# Design System – Jun-Oro v2

Bu doküman UI komponentleri ve sayfalar için görsel standartları tanımlar.

## Renk Paleti (Claude-inspired)
- Background: `#F5F3EE`
- Card: `#EEEAE4`
- Text Primary: `#2D2A26`
- Text Secondary: `#6B6661`
- Accent: `#D97757`

## Neumorphism Shadows
- Outer: `5px 5px 10px rgba(0,0,0,0.1), -5px -5px 10px rgba(255,255,255,0.7)`
- Inset: `inset 2px 2px 5px rgba(255,255,255,0.5), inset -2px -2px 5px rgba(0,0,0,0.1)`

## Spacing Scale
- Base: 8px
- Standard: 16px, 24px, 32px
- Large: 48px, 64px

## Border Radius
- Default: 16-20px
- Buttons: 12px
- Small elements: 8px

## Typography
- Font: Inter
- Title: 20-24px, weight 600
- Body: 14-16px, weight 400
- Line height: 1.5

## Animation
- Duration: 300-500ms (subtle)
- Easing: ease-in-out
- Hover: Lift (-4px) + Glow

## Desktop Resolutions
- 1920x1080 (base)
- 2560x1440
- 2560x1080 (ultrawide)
- 3440x1440 (ultrawide)
